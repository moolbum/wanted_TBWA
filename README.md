## 프로젝트 실행 방법

1. git clone
   ```bash
   git clone https://github.com/wanted-freepass/wanted_TBWA.git
   ```
2. 패키지 설치
   ```bash
   npm install
   ```
3. local 프로젝트 실행
   ```bash
   npm start
   ```
